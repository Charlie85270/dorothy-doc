## v1.2.6

<sub>March 28, 2026</sub>

- Fixed delay between PTY write and carriage return for Telegram/Slack input
- Added Chrome browser sharing for agents (`--chrome` flag)
- Fixed CLI path detection, settings sidebar scroll, and UX improvements

## v1.2.5 — Major Release

<sub>March 17, 2026</sub>

**New Agent Providers**
- Support for **PI** agent provider
- Support for **OpenCode** agent provider

**Integrations**
- Support for JIRA self-hosted domains

**Menu Bar**
- macOS menu bar tray with live agent status panel
- Status tabs: Working, Waiting for inputs, Ready to work, Idle
- Full-color Dorothy logo in menu bar

**Agent Management**
- External MCP server management from settings
- Live task preview next to agent name
- Revamped agents page with improved layout and filtering
- Status line with model, context usage, git branch, session time, and token stats

**Notifications**
- Custom MP3/audio file support per notification type
- New "Response Finished" notification toggle

**Hooks**
- Dedicated `PermissionRequest` and `TaskCompleted` hook events

**Fixes and Improvements**
- Fixed agent status lifecycle
- Pinned and favorites projects
