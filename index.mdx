## What is Dorothy?

Dorothy is a free, open-source desktop application that lets you **run and manage multiple AI coding agents simultaneously** from a single interface. Instead of juggling multiple terminal windows, Dorothy gives you a unified dashboard to orchestrate your entire AI agent fleet.

<Note>
  Dorothy is currently available for **macOS**. Windows and Linux support is coming soon.
</Note>
